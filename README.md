## 📖 Overview
tech-digest is a single-file Python script that pulls the latest buzz from the tech ecosystem. It retrieves the top 20 trending GitHub repositories, the newest 10 AI papers from arXiv, and the most popular posts from Hacker News and Reddit side-project threads. The output is a Markdown digest saved to a file, or sent directly to a Telegram chat if a bot token and chat ID are supplied. It is aimed at developers, product managers, and growth teams who need a quick, zero-config overview of emerging trends.
